Conversor de Divisas

Este proyecto consiste en una aplicación web para convertir divisas utilizando tasas de cambio actuales obtenidas desde una API.
La aplicación permite realizar conversiones entre dólares estadounidenses (USD), pesos mexicanos (MXN) y euros (EUR).

Tecnologías utilizadas:

1. HTML
2. CSS
3. JavaScript

Uso de la aplicación:

1. Ingresar la cantidad a convertir.
2. Seleccionar la moneda de origen.
3. Seleccionar la moneda de destino.
4. Presionar el botón Convertir.
5. Visualizar el resultado obtenido.
